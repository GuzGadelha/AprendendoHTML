Usando links externos

Você pode acessar o meu repositório do GitHub aqui!!!

Voê tambem pode escutar uma musica boa clicando aqui!!!

Observe que ao clicar no link do github é aberta uma nova aba no seu navegador, e ao clicar no link do youtube a pagina é sobreposta, isso ocorre pelo uso dos parâmetros target="_blank"

Observe também o uso de rel="external", este serve para indicar o destino para o link, temos 4 complementos para rel, são eles:

  • next: indica que o link é para a próxima parte do documento atual
  • prev: indica que o link é para a parte anterior do documento atual
  • author: indica que é um link para o site do autor do artigo atual
  • external: indica que é um link para outro site que não faz parte do site
