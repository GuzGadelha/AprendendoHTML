Principais formatações

Negrito (Destaque)

Nessa frase temos um termo em negrito usando a tag <b> (teg não semântica)

nessa frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico (ênfase)

nessa frase temos um termo em italico usando a tag <I> (não semântica)

nesta frase temos um termo em italico usando a tag <em> (semântica)

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e pequeno usando a tag <small>

A tag big não é mais utilizada pois é de entendimento que letras grandes são parte do estilo e estilo faz parte do css

Já a tag small ainda é usada, por exemplo em contratos fala-se de ler as letras pequenas, ou seja, tem significado semântico

Texto deletado

Podemos marcar um texto como excluido para marcar que ele deve ser lido mas não considerado, usando a tag <del>

Texto inserido

podemos marcar um texto como inserido usando a tag <ins> para dar uma enfase e indicar que ele foi adicionado depois

Podemos sublinhar com <U> entretanto não terá valor semântico

Texto sobrescrito

Para inserir coisas do tipo x20 + 3
Use a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O
Use a tag <sub>
